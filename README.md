# Order-Trace

Log in using the order-number to access the order progress

Getting Started with Create Vite App

TypeScript + React

# Setup

yarn global add json-server
copy .env.example to .env.local

### `yarn install` 

## Run server Command: 
### `yarn mock-api`

## Run Project 
### `yarn dev`
